神秘模块入门教程（Android端）

发表于2023-04-22|更新于2023-04-22|教程
|浏览量:

使用前提
1.已经安装Magisk或KernelSU的安卓设备

2.安卓6.0以上

一 安装node

下载Tetmux

https://github.com/termux/termux-app/releases

默认下第2个，其他根据自己的系统架构选择

设置国内镜像

命令行输入

apt更新所有包

apt update && apt upgrade

看到输入Y

重新设置国内镜像

termux-change-repo

安装node

apt install nodejs

看到输入Y

检查node

node -v

返回版本号就安装成功了

第二步 模块刷入

在发布页下载最新的神秘模块 一般其名称应为 神秘_android_发布时间.zip

TG群组链接：https://t.me/blowh2o

123云盘下载链接：https://www.123pan.com/s/bAORVv-8XQl.html

安装模块

进入面板

打开127.0.0.1:23333

配置神秘

第一步 选择 涂涂改改

第二步 选择星盘

新建星盘

选召唤类星盘

填 星盘名称 召唤法阵其他看需求的

第三步 在涂涂改改-传送阵

新建传送阵

选择定向司南

选择创建的星盘

第四步 自动测速（非必选）

新建传送阵

选择随缘司南

选择想自动测速的星盘

第五步 传送阵套娃

根据传送阵选择合适的传送阵套娃

国外选择proxy，国内选择domestic

选择你想套娃的传送阵

第六步 更新

选择你想更新的星盘当然可以全部更新

提示处理完成就成功更新了

第七步 使用

选择查星盘

左边选择代理

找到传送阵套娃名称选传送阵最好自动测速

第三步 国内外分流&基础正则使用

注:一般一个机场都是国内国外节点混合，我们要用正则进行过滤

1.新建传送阵

注:名称不要同名

传送阵类型选择定向司南

选择你要筛选的星盘

例:cn|深圳 筛选CN和深圳的节点名称

“|”进行分隔，切记请不要“|”后面空白，否则将筛选全部

2.自动测速（非必选）

新建个传送阵，传送阵类型选择随缘司南

正则如上

选择你要筛选的星盘

注:只选填红框内，其他无管

根据传送阵选择合适的传送阵套娃

国外选择proxy，国内选择domestic

选择合适的传送阵

第四步 热点模式

热点模式和兼容模式打开

第五步 常见问题（非必选）
